Grid

Create a multi-column, dynamic grid layout whose items can be sorted and filtered.

The Grid component allows you to create a dynamic and responsvie grid layout utilizing the Grid component. Grid items will arrange themselves fluently and seamlessly for a gap-free multi-column layout on all device sizes.

Usage

To apply this component, add the data-uk-grid attribute to the container element. Set the width of the grid items by using the uk-width-* or .uk-grid-width-* classes from the Grid component.

<!-- Example using uk-width-* -->
<div data-uk-grid>
    <div class="uk-width-small-1-2 uk-width-medium-1-4">...</div>
    <div class="uk-width-small-1-2 uk-width-medium-1-4">...</div>
</div>

<!-- Example using uk-grid-width-* -->
<div class="uk-grid-width-small-1-2 uk-grid-width-medium-1-4" data-uk-grid>
    <div>...</div>
    <div>...</div>
</div>

Filtering

You can also filter your grid so that only particular items will be displayed. To do so, add the {controls: '#my-id'} option to the data-attribute to target the id for the filter controls. Each control needs to have the data-uk-filter attribute to define the category you want to filter. Then you need to add the data-uk-filter attribute also to each grid item, to define in which category the item is in.

Example

  • Filter A
  • Filter B
  • Reset
A 1
B 2
A 3
B 4
B 5
A 6
B 7
A 8

Markup

This example is using a Subnav to filter the items.

<!-- Filter Controls -->
<ul id="my-id" class="uk-subnav">
    <li><a href="" data-uk-filter="filter-a"></a></li>
    <li><a href="" data-uk-filter="filter-b"></a></li>
    <li><a href="" data-uk-filter=""></a></li>
</ul>

<!-- Dynamic Grid -->
<div data-uk-grid="{controls: '#my-id'}">
    <div data-uk-filter="filter-a">...</div>
    <div data-uk-filter="filter-b">...</div>
</div>

Sorting

Grid items can be sorted ascendingly or descendingly by number or character. To do so, add the {controls: '#my-id'} option to the data-attribute to target the id for the sorting controls. Each control needs to have the data-uk-sort="order" or data-uk-sort="order:desc" attribute, depending on whether ther ordering is ascending or descending. Then you need to add the data-order attribute to each grid item and targetting an alphabetic or numeric string of each item.

Example

In this example items are ordered by numericly as well as alphabetically.

  • Character (Asc)
  • Character (Desc)
  • Number (Asc)
  • Number (Desc)
A 8
B 7
C 6
D 1
E 5
F 3
G 2
H 4

Markup

<ul id="my-id" class="uk-subnav">
    <li><a href="" data-uk-sort="order"></a></li>
    <li><a href="" data-uk-sort="order:desc"></a></li>
    <li><a href="" data-uk-sort="order-number"></a></li>
    <li><a href="" data-uk-sort="order-number:desc"></a></li>
</ul>

<div data-uk-grid="{controls: '#my-id'}">
    <div data-order="a" data-order-number="8">...</div>
    <div data-order="b" data-order-number="7">...</div>
</div>
